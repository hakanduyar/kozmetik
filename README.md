# Frontend Test Case – Kozmetik E-Ticaret Uygulaması HAKAN DUYAR - iamhakanduyar@gmail.com

## Kullanılacak Teknolojiler

Bu test kapsamında aşağıdaki teknolojiler kulanılacaktır:
